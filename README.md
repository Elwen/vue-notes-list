# vue-notes-list

SPA приложение для заметок.

## Установка зависимостей

```sh
npm install
```

###  Компиляция и запуск в режиме разработки

```sh
npm run dev
```

### Сборка приложения

```sh
npm run build
```

### Lint с [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Форматирование кода с [Prettier](https://prettier.io/)

```sh
npm run format
```
